# 🌦️ Java Weather App

A simple desktop Weather Application built using Java Swing and OpenWeather API.

## 🚀 Features
- Real-time weather data
- City search functionality
- Weather, Temperature, humidity
- Clean GUI interface

## 🛠️ Tech Stack
- Java
- Swing (GUI)
- OpenWeather API

## 📂 How to Run
1. Open in any Java IDE (Eclipse / IntelliJ)
2. Add JSON library (org.json)
3. Add your API key
4. Run WeatherApp.java
